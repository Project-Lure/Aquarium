Creator

「ぎじえプロジェクト / Project LURE」を制作している人についてのページです。
プロフィールや創作のスタンスなど、必要な範囲だけを簡潔にまとめます。

プロフィール

ハンドルネーム、活動名義、主な活動領域（イラスト / 作曲 / 小説 / デザイン など）を、 公開しても問題ない範囲で記載する想定です。

創作のスタンス

作品との距離感、得意なジャンル、こだわっているポイントなどを整理します。
「キャラデザを軸にIP全体を組み立てる」「色と物語の対応関係を大事にしている」など、 自分用のメモにもなります。

Project LUREに込めているテーマ

個人的な経験や感情から、このプロジェクトにどんなテーマを反映させているのかを書いておくスペースです。
重くなりすぎない範囲で、「こういう視点で作品を見てもらえると嬉しい」といった補足を書く想定です。
