Terms / 用語説明

Project LURE の中で頻出する専門用語や略語を、
まとめて確認できる用語集です。

情報が増えてきたら、ここを見れば最低限の意味が追えるように整えていきます。

ARC / アーク

アーク（Arc）は、キャラクターの内面や役割を象徴する属性ラベルです。
戦闘の有利不利を決めるゲーム的な属性ではなく、 そのキャラがどんな方向の「物語」を持っているかを整理するための分類として使います。

🔥 ブレイズ（Blaze）
情熱 衝動 燃え広がる

※ 実際のアーク一覧・絵文字セット・略号（B / F / G …など）は、
データが固まり次第ここに追記していきます。

CODE / コード

各キャラクターには、3桁の英数字コードが割り振られます。
例：000 / 004 / 40A など。

コードは、
「どのシリーズに属するか」「その中で何番目のキャラクターか」などを
構造的に表すために使います。

そまりものがたり（0_Occupation）など、シリーズごとのコードルールは、
ある程度固まった段階でここにまとめていきます。

SERIES / シリーズと総称

Project LURE の中には、いくつかの大きなシリーズ（大テーマ）が存在します。
例：

  • 0_Occupation …… そまりものがたり
  • 1_Instinct …… こころものがたり
  • 2_Zone …… あそびものがたり（仮）

ここには、「日本語タイトル」「英語タイトル」「略称」などを一覧で整理していきます。
