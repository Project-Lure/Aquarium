Project / ぎじえプロジェクトとは

「ぎじえプロジェクト / Project LURE」は、
色から生まれたキャラクターの “生き方や心の揺れ” を、キャラデザとキャラソンを中心に描く創作企画です。

物語を長編で語るのではなく、キャラの人生観に触れられる小さな物語や歌を積み重ねて世界が見えていく形式をとっています。

CONCEPT

Project LURE の軸は「色彩 × 物語」。
ひとつの色を起点に、コード・テーマ・メインカラーを設定し、 その世界で生きるキャラクターが形づくられます。そして、キャラクターの悩みや葛藤を描きます。

キャラクターの心情や価値観は、キャラソンという“声”の形で表現されます。
それぞれの曲は、そのキャラがどんな気持ちで世界を見ているのかを示します。

SERIES

Project LURE の世界には、いくつかのテーマ別シリーズがあります。
職業を扱う 0_Occupation / そまりものがたり、感情を扱う 1_Instinct / こころものがたり など、テーマに応じて分類されています。

シリーズは世界線そのものではなく、作品を整理するためのラベルとして機能します。

シリーズや分類の詳細は、
Terms / 用語説明 にまとめています。

GOAL & FUTURE

まずは、キャラクターのビジュアルカードと、対応する キャラソンを順に制作し、作品として公開していくことを目標にしています。

キャラソンは世界観の核となる要素であり、今後は作品展開の中心として扱っていきます。
キャラクターは単発で終わらせず、展示・グッズ・コラボなどにも展開できる IP（キャラクター資産）として育てていきます。
