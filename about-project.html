Project

「ぎじえプロジェクト / Project LURE」全体のコンセプトをまとめたページです。
なぜ「色」と「キャラクター」を組み合わせているのか、このプロジェクトでやりたいことは何か、 といった大枠の話をここに集約します。

プロジェクトのコンセプト

ここに、「色×キャラクター×物語×キャラソン」という軸の説明を入れる想定です。
・色からキャラクターが生まれること
・キャラクターの物語がイラストや楽曲に落とし込まれること
・単発ではなくシリーズとして世界を広げていくこと
など、全体の方針を整理して書いておきます。

シリーズ構成

0_Occupation（そまりものがたり）を起点に、1_Instinct、2_Zone … と続いていく、 シリーズ全体の「棚」のイメージを説明します。
それぞれどんなテーマを扱うシリーズなのか、簡単な箇条書きで載せておくと見やすいです。

このプロジェクトで目指していること

展示会やグッズ、配信、アルバム制作など、将来的にやりたいことの方向性を書いておく想定です。
「こういう形で作品に触れてもらいたい」「最終的にはここまで行きたい」といった、 ビジョンのメモ置き場として使えます。
